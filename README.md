# Boba Shop Accounting Sim
#### Track the sales and expenses of a boba tea shop called Boba Tea Shop.

> Fun fact: "Boba Tea Shop" is an instance of a boba tea shop, btw.


## How to Use
When launching the application, the homescreen will be displayed.

From there, you can choose between recording a sale, recording an 
expense, or opening the ledger screen. 

In the ledger screen, you can choose from various transaction types to display, or you can open the "reports screen"
